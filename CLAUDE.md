# CLAUDE.md

## Project: Rawey Perfume Store

### Brand
- Rawey is a luxury perfume brand.
- The visual identity is soft luxury: ivory, warm white, muted gold, champagne, beige, and deep black accents.
- Leave a clean empty space for the logo in the header and hero. Do not generate a new logo.

### Product Philosophy
- Everything should feel premium, editorial, calm, and expensive.
- Copy must be short, confident, elegant, and brand-like.
- Avoid generic ecommerce or template-looking UI.

### Tech Stack
- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion
- lucide-react
- Local mock data only unless explicitly asked otherwise

### Design Rules
- Use generous whitespace.
- Prioritize strong typography and visual hierarchy.
- Use soft shadows, subtle borders, smooth transitions, and refined hover states.
- Keep the palette restrained.
- Make the site look cinematic and high-end, not busy.

### Required Pages
- Home
- Shop
- Category pages
- Product details
- About
- Contact
- Cart
- Checkout
- Wishlist
- FAQ
- Search results

### Required Interactions
- Search
- Category filters
- Price filters
- Sort controls
- Add to cart
- Remove from cart
- Quantity adjustments
- Wishlist toggles
- Navigation between sections and pages
- Empty states and success states

### Data
- Use believable mock products and collections.
- Include fields like name, collection, price, size, notes, rating, availability, season, occasion, and scent family.
- Make the data feel like a real perfume brand catalog.

### taste-skill
- Use taste-skill as the design reference for premium frontend output.
- Favor refined layout, typography, spacing, color harmony, and motion.
- Prefer the most elegant solution over the simplest generic one.

### Workflow
- Plan briefly.
- Build the structure.
- Implement the design system.
- Add all pages and interactions.
- Polish spacing, motion, and responsiveness.
- Verify routes, buttons, and states.
- Fix errors before finishing.

### Quality Bar
- No placeholder comments.
- No half-finished sections.
- No TODOs left behind.
- No generic template feel.
- No broken links or dead UI.

### General Instruction
If a design choice is ambiguous, choose the option that makes Rawey look more luxurious, more credible, and more editorial.
